Colophon

A perfect front-end template to start writing your own colophon. It's simple, just fill in the blanks!

Download this project as a .zip file Download this project as a tar.gz file

This is my attempt to thank every single individual who knowingly or unknowingly contributed to the development of this website. To checkout the colophon of this website, simply toggle the blanks.

Typography

Primary fonts used extensively across this website are Chaparral Pro and Skolar Web embedded by @font-face rule and served via Typekit. Fallback for the primary fonts are Palatino, Georgia, Times New Roman fall under the category of serif fonts. To maintain vertical rhythm, the baseline is roughly configured to 24px with the help of Em Baseline Generator developed by Joshua Hibbert and then verified by Basehold.it and GridWax.

Colors

Color schemes are heavily based on the Android Color Swatches. The color contrast ratio is measured with the help of a neat little tool made by Lea Varou. Colors in the stylesheet are embedded in the hsla() format. ColorHexa is always my best friend to find out the closest web safe color.

Framework

HTML5 Boilerplate is the framework behind this website. Coding standards are based on the NA Isobar Coding Standards and the book Dive Into HTML5 by Mark Pilgrim has always been a source of inpiration for me.

Images

All the images are individually smushed using Smush.it and sprited using the SpritePad and then compressed again. The SVG Icons are brought to you by The Noun Project and the logo for personal branding is done by Abhjijeet Wankehede. Images and SVG Icons are embedded directly via CSS using base64 encoded Data-URI generated using Image to Data URI Converter by Web Semantics. All the images are manufactured by Adobe Photoshop CS6 as 24-bit PNG image with full alpha channel transparency. Subsequently these images are then injected through TinyPNG to compress them into 8-bit indexed color images with 100% support for transparency.

Plugins

This site currently doesn't use any jQuery plugin but if you are in search for plugins, then UnHeap and JSDB.io are always a good place to start.

Stylesheet

CSS code is formatted is using ProCSSor and compressed using the Online YUI Compressor. CSS reset is based on the reset.css of Eric Meyer for old and legacy browsers, while modern browsers rely on normalise.css by Nicolas Gallagher. Helium-css has been extensivley used to churn out the unused css across pages. CSS code is verified for errors using CSS Lint and W3C CSS Validator. Special unicode characters are brought to you by Copy-Paste Character.

Scripts

All the JavaScript files are compressed using Uglify.js and YUI Compressor. Performance optimisation is carried out with the help of jsPerf

Design

Initial layout for the website was designed and conceived using wireframing technique with the help of a neat little free app - Wireframe.cc. Charts and Infographics are brought to you by xCharts and Chart.js.

Browser

Compatible with the latest versions of all modern browsers and tested with BrowserStack for both desktop and mobile. A/B testing done with Optimizely. Screenshots are generated with the help of PlaceIt by Breezi. Performance for mobile devices is measured with Mobitest and the test results are available here.

Hosting

This website is hosted on GitHub and version controlled by Git. The domain is brought to you by BigRock and Email with personal domain was made possible by Google Apps.

Desk

Completely handcrafted and coded in Sublime Text 3 on 13-inch Retina MacBook Pro. The Editor uses the Soda theme and the dock icon is from Elliot Jackson. Early versions of this website were coded in VIM. H5BP Ant Build Script is used to concatenate and minify the assests and make it production ready.
